# 解封

> 百科 Key: `战斗-攻击与防御-解封`
> 层级: 战斗 / 攻击与防御 / 解封

解封是[查:铸剑山庄]功法特有的攻击方式。

释放解封攻击时，将对敌人造成不可避免的大量伤害，并根据兵器类型产生不同的效果。

**各类型兵器解封效果一览**

| 兵器类型 | 解封效果 |
| --- | --- | 
| 剑 | 攻击3次，每次将敌人的1个随机功法[查:封禁]5秒 |
| 刀 | 攻击3次，对敌人造成的直接伤势变化为更难恢复的[查:旧伤] |
| 长兵 | 攻击3次，[查:攻击范围]变化为最大 |
| 掌套 | 攻击3次，每次吸取敌人20%的随机真气 |
| 暗器 | 攻击3次，分别对敌人造成1~3级的[查:破绽] |
| 药霜、毒砂 | 攻击3次，施加的毒素为兵器的10倍 |
| 暗器、针匣、机关 | 攻击3次，分别对敌人造成1~3级的[查:封穴] |
| 拂尘、长鞭 | 攻击3次，消除敌人当前运起的身法 |
| 对刺、短杵、令符 | 攻击3次，消除敌人当前运起的护体功法 |
| 箫笛、瑶琴 | 攻击3次，每次使敌人的[查:内息紊乱]+30 |


铸剑山庄剑法、刀法、长兵功法的心法效果，会在战斗中为运用者装备的所有对应类型的兵器持续增加解封进度。

当兵器的解封进度达到100%时，即可释放此兵器的解封攻击。释放解封攻击后，兵器的解封进度重置为0%。

下三品/中三品/上三品的功法，每秒增加的解封进度分别为1.33%/2.00%/2.66%。同一类型的多个功法增加的进度不会叠加生效，只取其中的最大值。

除了解封剑、刀、长兵外，还可以通过铸剑山庄部分功法的心法效果，或[查:天枢玄铸]机关人的助战指令「解封」，增加其他类型兵器的解封进度，以释放解封攻击。

- 解封攻击不会被敌人[查:化解]，但也无法获得[查:架势]和[查:蓄式]。

- 以解封攻击造成[查:直接伤害]时，必然[查:命中要害]。

- 当双方距离不在兵器的[查:攻击范围]内时，无法释放解封攻击。

- [查:临机兵器]不会增加解封进度，也无法释放解封攻击。


解封攻击威力计算公式：

解封攻击威力 = 100% + 兵器威力 ÷5


### 生铸

生铸是[查:铸剑山庄]长兵功法特有的效果。

铸剑山庄长兵功法的心法效果，可以在运用者解封长兵时，将一件装备变为生铸神器，并赋予生铸特性。

如果生铸神器的品级高于原装备，则根据功法效果，需要额外消耗[查:精制材料]才能生铸。


生铸神器的特点：

- 生铸神器拥有对应功法所赋予的生铸特性，效果见下表。

- 生铸神器的类型、材质与原装备相同，并且继承原装备的[查:装备特性]、[查:精制]效果和[查:淬毒]状况。

- 生铸神器的耐久每10秒减少一次，耐久为0时失去生铸特性，变回原装备。

- 战斗结束后，所有生铸神器都将失去生铸特性，变回原装备。

[查:架海神杖]的生铸效果仅为敌人的装备添加生铸特性，不会将对应装备变为生铸神器。

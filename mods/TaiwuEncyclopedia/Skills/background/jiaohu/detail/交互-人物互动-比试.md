# 比试

> 百科 Key: `交互-人物互动-比试`
> 层级: 交互 / 人物互动 / 比试

通过比试互动，可以与他人进行战斗、较艺或促织决斗。


### 促织决斗

互动内容：与对方进行促织决斗。关于促织决斗的具体规则，详见「[查:促织决斗]」条目。

使用需求：对方好感为冷淡及以上；太吾持有至少3只存活的促织

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 若太吾胜利，则赢得对方的押注；若太吾失败，则失去自己的押注。

- 若太吾胜利，则对方好感上升，心情降低；若太吾失败，则对方好感降低，心情上升。


### 较艺比试

互动内容：与对方进行较艺。关于较艺的具体规则，详见「[查:较艺]」条目。

使用需求：对方好感为热忱及以上

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 与对方进行较艺。

- 若太吾胜利，则对方好感上升，心情降低；若太吾落败，则对方好感降低，心情上升。


### 切磋武功

互动内容：向对方发起点到为止的切磋，双方以[查:切磋]的方式进行战斗。

使用需求：对方好感为热忱及以上

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 与对方进行切磋。

- 若太吾胜利，则对方好感上升，心情降低；若太吾落败，则对方好感降低，心情上升。


### 发起挑战

互动内容：向对方发起真刀真枪的较量，双方以[查:恶斗]的方式进行战斗。

使用需求：对方好感为热忱及以上；对方[查:精纯]不低于太吾

使用限制：每月对同一人物限用一次

消耗精力：3

互动效果：

- 与对方进行恶斗。

- 若太吾胜利，则对方好感上升，心情降低；若太吾落败，则对方好感降低，心情上升。


比试互动效果公式：

如果太吾胜利，对方的好感和心情根据其立场变化：

- 对方好感：刚正 = +600；仁善 = +1200；中庸 = +600；叛逆 = +1200；唯我 = +600

- 对方心情：刚正 = -2；仁善 = -3；中庸 = -2；叛逆 = -3；唯我 = -2

如果太吾失败，对方的好感和心情根据其立场变化：

- 对方好感：刚正 = -1200；仁善 = -600；中庸 = 0；叛逆 = -1200；唯我 = -600

- 对方心情：刚正 = +2；仁善 = +3；中庸 = +2；叛逆 = +3；唯我 = +2

如果太吾在切磋、挑战中逃跑，且对方立场为刚正，则对方好感额外-1000。

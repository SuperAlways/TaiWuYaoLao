# 助战指令

> 百科 Key: `战斗-指令与行为-助战指令`
> 层级: 战斗 / 指令与行为 / 助战指令

在战斗中，人物可以通过助战指令，指挥各自的[查:助战同道]协同作战。


每名人物最多拥有3个助战指令。

月份更替时，人物的助战指令会根据其各项[查:特性倾向]的数量发生变化。

太吾可以为自己的同道更换助战指令。更换后，同道的助战指令将不再随月份更替而变化。

更换助战指令时，各项指令需要的特性倾向之和不能超过人物拥有的特性倾向。

完成狮相门的地区故事后，可以通过[查:统筹方略]，强化同道的助战指令。

**助战指令一览**

| 助战指令 | 特性倾向 | 效果 | 冷却所需进度 | 跳过冷却需要好感 |
| --- | --- | --- | --- | --- | 
| 攻击 | 进攻×1 | 以兵器攻击敌人。 | 36000 | ≥亲密 |
| 掠阵 | 进攻×1 | 在30秒内以自身力道、精妙、迅疾、动心的40%，增加主战者的对应属性。 | 360000 | ≥喜爱 |
| 摧破 | 进攻×2 | 施展摧破功法攻击敌人，此功法的威力增加30%。 | 720000 | ≥亲密 |
| 催发 | 进攻×3 | 使主战者功法的施展进度增加30%。 | 72000 | ≥喜爱 |
| 出战 | 进攻×4 | 在20秒内代替主战者进行战斗。期间玩家可以操控此同道，且此同道的攻击、防御、命中、化解属性均提高20%。 | 720000 | ≥亲密 |
| 牵制 | 守御×1 | 在15秒内阻止敌方同道使用助战指令。 | 360000 | ≥喜爱 |
| 警示 | 守御×1 | 在30秒内以自身卸力、拆招、闪避、守心的20%，增加主战者的对应属性。 | 360000 | ≥喜爱 |
| 还护 | 守御×2 | 为主战者消除2个破绽。 | 36000 | ≥喜爱 |
| 解穴 | 守御×3 | 为主战者解开2个封穴。 | 36000 | ≥喜爱 |
| 防御 | 守御×4 | 施展护体功法为主战者抵挡敌人的攻击，此功法的威力增加30%。 | 180000 | ≥亲密 |
| 疗伤 | 机略×1 | 为主战者治疗伤势。 | 72000 | ≥喜爱 |
| 驱毒 | 机略×2 | 为主战者驱除毒素。 | 72000 | ≥喜爱 |
| 真气 | 机略×3 | 将自身的各类真气各5点传输给主战者。 | 72000 | ≥亲密 |
| 迅进 | 机略×4 | 将双方的距离拉近1。 | 240000 | ≥喜爱 |
| 拉回 | 机略×4 | 将双方的距离拉远1。 | 240000 | ≥喜爱 |


在战斗中使用助战指令后，此指令需要经过一段冷却时间才能再次使用。不同助战指令的冷却时间不同。助战同道对主战者的好感越高，助战指令的冷却时间越短。

如果助战同道对主战者的好感达到一定程度，其助战指令每次使用后，将有25%的概率跳过冷却，可以直接再次使用。


助战指令冷却时间计算公式：

冷却时间（帧） = 指令冷却所需进度 ÷ 每帧增加进度

- 每帧增加进度：由同道对主战者的好感决定。

  憎恨及以下 = 10；达到敌视 = 25；达到陌路 = 40；达到冷淡 = 70；达到热忱 = 85；达到亲密 = 100

- 每60帧为1秒


### 负面指令

战斗开始时，若助战同道对主战者的好感为鄙视及以下，助战指令会被随机替换为负面指令。

在战斗中，助战同道会自动随机使用负面指令，对己方主战者造成不利影响。

若好感为鄙视或敌视，出现1个负面助战指令；若好感为仇视或憎恨，出现2个负面助战指令；若好感为痛恨或血仇，出现3个负面助战指令。

| 助战指令 | 特性倾向 | 效果 |
| --- | --- | --- | 
| 打断 | 守御 | 因心生嫌隙，可能在主战者[查:施展功法]时，伺机将主战者的施展打断。 |
| 陷害 | 机略 | 因心生嫌隙，可能在主战者不在敌人的[查:攻击范围]内时，伺机将主战者向敌人的攻击范围内推拉。 |
| 偷袭 | 进攻 | 因心生嫌隙，可能伺机偷袭主战者，对主战者进行[查:兵器攻击]，并造成[查:破绽]、[查:封穴]。 |
| 散功 | 守御 | 因心生嫌隙，可能伺机消除主战者持续中的[查:护体功法]、[查:身法]。 |
| 暗害 | 进攻 | 因心生嫌隙，可能伺机以医术、毒术对主战者造成伤势、毒素。 |
| 纠缠 | 进攻 | 因心生嫌隙，可能伺机在短时间内，降低主战者的所有命中属性和化解属性。 |
| 扰乱 | 守御 | 因心生嫌隙，可能在主战者进行各项[查:战斗行为]时，伺机将行为打断。 |
| 散气 | 机略 | 因心生嫌隙，可能伺机消散主战者的真气。 |

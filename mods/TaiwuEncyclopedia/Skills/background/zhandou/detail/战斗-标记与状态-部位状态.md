# 部位状态

> 百科 Key: `战斗-标记与状态-部位状态`
> 层级: 战斗 / 标记与状态 / 部位状态

在战斗中，人物的各个身体部位可能会出现各种负面状态。


### 破绽

当人物使用[查:变招]或[查:摧破功法]命中敌人后，有可能使敌人的对应身体部位出现破绽，并产生[查:破绽标记]。

出现破绽的部位受到的直接伤害提高40%，同部位的多个破绽效果不会叠加。


破绽分为1~3三个等级。

- 以不同兵器变招造成的破绽等级可能不同。

- 摧破功法的品级越高，攻击造成的破绽等级越高。[查:下三品]/[查:中三品]/[查:上三品]功法造成的破绽分别为1/2/3级。

- 某些功法的效果可能将破绽等级提升至4级。


每个破绽会在出现一段时间后消失。等级越高的破绽消失得越慢。

每个身体部位最多存在3个破绽。当破绽数量达到上限后，新出现的破绽会替换等级更低的破绽。

战斗结束后，人物身上的所有破绽将会移除。


### 封穴

当人物使用[查:变招]或[查:摧破功法]命中敌人后，有可能使敌人的攻击部位出现封穴，并产生[查:封穴标记]。

不同部位的封穴将对人物造成不同的妨害效果，同部位的多个封穴效果不会叠加。

当一个部位存在3个封穴时，人物将无法施展需要此部位的功法。

**各部位封穴效果一览**

| 部位 | 封穴效果 |
| --- | --- | 
| 头颈 | 头颈封穴会减少人物至少0.4、至多0.8的[查:攻击范围] |
| 胸背 | 胸背封穴会使人物[查:提气]的恢复速度降低至少20%、至多60% |
| 腰腹 | 腰腹封穴会使人物[查:架势]的恢复速度降低至少20%、至多60% |
| 左臂 | 左臂封穴会使人物进行兵器攻击的[查:攻击准备]时间提高至少50%、至多150% |
| 右臂 | 右臂封穴会使人物进行兵器攻击的[查:攻击准备]时间提高至少50%、至多150% |
| 左腿 | 左腿封穴会使人物进行[查:移动]所需的时间提高至少50%、至多150% |
| 右腿 | 右腿封穴会使人物进行[查:移动]所需的时间提高至少50%、至多150% |


封穴分为1~3三个等级。

- 以不同兵器变招造成的封穴等级可能不同。

- 摧破功法的品级越高，攻击造成的封穴等级越高。[查:下三品]/[查:中三品]/[查:上三品]功法造成的封穴分别为1/2/3级。

- 某些功法的效果可能将封穴等级提升至4级。


每个封穴会在出现一段时间后消失。等级越高的封穴消失得越慢。

每个身体部位最多存在3个封穴。当封穴数量达到上限后，新出现的封穴会替换等级更低的封穴。

战斗结束后，人物身上的所有封穴将会移除。


### 重伤

当一个身体部位的内伤或外伤达到5层时，此部位处于重伤状态。

当人物施展功法时，若[查:施展部位]处于重伤状态，则此功法的[查:施展时间]加倍。


### 残毁

当一个身体部位的内伤或外伤达到6层时，此部位处于残毁状态。

人物无法施展需要残毁部位的功法。


### 剧痛与伤坏

[查:血犼教]拳掌、指法的心法效果，可以对人物的身体部位造成剧痛或伤坏。

剧痛或伤坏的效果以人物特性的方式呈现。

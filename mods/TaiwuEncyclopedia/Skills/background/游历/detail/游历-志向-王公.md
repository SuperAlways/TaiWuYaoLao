# 王公

> 百科 Key: `游历-志向-王公`
> 层级: 游历 / 志向 / 王公

*身如列鼎之贵，势如鸣朝之尊，黔黎等闲不敢言，而我独上青云霄……*


志向造诣：品鉴

志向衣装：[查:衮龙黄袍]


### 资历获取

王公志向资历获取方式：

- 促织：通过[查:捕捉促织]或封侯拜相获得促织；促织[查:决斗战绩]中的可赞胜利增加

- 奇遇：在「促织大会」[查:奇遇]中取得称号「促织将军」

- 志向技能：使用鉴别促织、百蜕仙灵


王公志向资历获取公式：

- 通过捕捉促织或封侯拜相获得促织时：增长资历 = 促织的价值 ÷ 2

- 促织决斗战绩中的可赞胜利增加时：增长资历 = （战胜的促织品级 + 2）² × 500

- 参加促织大会并取得称号「促织将军」时：增长资历 = 307500

- 使用鉴别促织时：

  鉴别出神采非凡的促织：增长资历 = 10250

  令促织蜕变：增长资历 = 153750

- 使用百蜕仙灵时：增长资历 = 被复活促织的价值 ÷ 2


### 鉴别促织

技能等级：一级

解锁节点：王公资历达到10%


使用方式：主动使用

使用消耗：5精力、1000历练

使用冷却：1个月

技能效果：可鉴别[查:促织]是否为[查:神采非凡]。神采非凡的促织达到一定的净胜场后，可使其蜕变为异品促织王。

- 净胜场：促织的[查:决斗战绩]中，可赞胜利减去决斗败绩的数量。

- 王公资历越高，促织蜕变需要的净胜场数越少。

效果公式：

促织蜕变所需净胜场数：100-资历百分比 ÷ 2（向下取整）

促织蜕变时，将变为与自身[查:元色]相同的随机一种促织王。

**促织蜕变一览**

| 促织元色 | 可能成为促织王 |
| --- | --- | 
| 青 | 天蓝青、真三色、赤须 |
| 黄 | 真紫黄、油纸灯、跑马黄、两头枪 |
| 紫 | 玉尾、披袍轩甲 |
| 红 | 三太子、草三段、反生名、绣花针 |
| 黑 | 八败、铁弹子、头陀 |
| 白 | 三段锦、梅花翅、朱砂额、玉锄头、吹铃 |


### 封侯拜相

技能等级：二级

解锁节点：王公资历达到30%


使用方式：人物互动

使用消耗：10精力、2500威望

使用冷却：1个月

技能效果：授予门派人物官职，并获得其提供的[查:门派支持]。

- 被授予官职的人物将获得相应的[查:称号]，戒心-50000。

- 人物担任官职后，无论犯下任何罪行，都不会受到[查:处罚]。

- 每年秋季时，每名被授予官职的人物将向太吾进贡1只促织。

- 太吾可以随时与人物互动，罢免其官职。罢免官职将使此人的好感-30000、戒心+80000，并有概率成为[查:仇敌]。

效果公式：

罢免官职结仇概率：根据对方的立场，刚正 = 5%；仁善 = 0%；中庸 = 10%；叛逆 = 20%；唯我 = 15%


### 百蜕仙灵

技能等级：三级

解锁节点：王公资历达到75%


使用方式：主动使用

使用消耗：10精力、25000历练

使用冷却：6个月

技能效果：令一只已经死亡的促织起死回生。

- 促织的[查:蛰龄]变为0岁，并恢复所有耐久。


### 万民来朝

技能等级：四级

解锁节点：王公资历达到100%


使用方式：主动使用

使用消耗：5精力、5000威望

使用限制：每12个月

技能效果：在3格范围内的半数地格中召来可捕捉的促织，且[查:促织缘]+300。

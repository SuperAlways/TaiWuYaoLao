# 交换物资

> 百科 Key: `物品-物品与资源-交换物资`
> 层级: 物品 / 物品与资源 / 交换物资

太吾可以通过「交换物资」互动或[查:库房交换]，与其他人物、势力库房交换物品和资源。

计入双方的交换优势影响后，如果太吾提供的物资[查:价值]/[查:贡献值]总和不低于对方，即可完成本次交换。


### 交换优势

在交换物资时，双方的交换优势会额外提高各自物资的价值/贡献值。

对方的交换优势越低，太吾的交换优势越高，则太吾能够换取更多的物资。


对方的交换优势受到以下因素影响：

- 好感：对方的[查:好感]越高，交换优势越低。

- 戒心：对方的[查:戒心]越低，交换优势越低。

- 立场：双方[查:立场]相合时，对方的交换优势更低；双方立场相冲时，对方的交换优势更高。

- 名誉：太吾的[查:名誉]越高，对方的交换优势越低。

- 性别：与少林派人物和库房交换时，若太吾不为男性，对方的交换优势更高；与璇女派人物和库房交换时，若太吾不为女性，对方的交换优势更高。

- 库房：进行库房交换时，若库房物资匮乏，则库房的交换优势更高。


太吾可以通过以下方式提升自身的交换优势：

- 使用秘闻：根据对方立场使用相应的[查:秘闻]，可以提升太吾的交换优势，并消耗秘闻的使用次数。

  对方为刚正：使用主要人物为太吾的正面秘闻

  对方为仁善：使用主要人物为太吾的正面秘闻，或主要人物为对方的负面秘闻

  对方为中庸：使用秘闻无法产生影响

  对方为叛逆：使用任意负面秘闻

  对方为唯我：使用主要人物为对方的正面秘闻

- 消耗支持：与门派人物或库房交换物资时，消耗此门派人物的[查:支持]，可以提升太吾的交换优势。

- 消耗恩义：消耗对方所属地区的[查:恩义]，可以提升太吾的交换优势。单次交换中最多可消耗1000地区恩义。


其他影响太吾交换优势的因素：

- 用对方喜好的物品交换：增加太吾的交换优势。如果对方立场为叛逆，则改为减少太吾的交换优势。

- 用对方厌恶的物品交换：减少太吾的交换优势。如果对方立场为叛逆，则改为增加太吾的交换优势。

- 用与对方有关系的人物交换：增加太吾的交换优势。

- 用触犯对方[查:门派戒律]的物品交换：减少太吾的交换优势。

- 拿取品级超出当前[查:侵袭进度]的物品：减少太吾的交换优势。

- 拿取对方装备中的物品：减少太吾的交换优势。


完成交换条件：

与人物交换：太吾交换优势 × 太吾提供物资价值总和 ≥ 对方交换优势 × 对方提供物资价值总和

与库房交换：太吾交换优势 × 太吾提供物资贡献值总和 ≥ 库房交换优势 × 库房提供物资贡献值总和


交换优势计算公式：

对方交换优势% = （基础优势 + 好感影响 + 戒心影响 + 立场影响 + 名誉影响 + 性别影响 + 库房影响） × 玄狱挑战影响

- 基础优势：非门派人物 = 200；门派人物 = 300；库房 = 400

- 好感影响：对方好感为不渝~血仇时，好感影响 = 0/0/0/0/0/25/50/100/200/350/550/700/1000

- 戒心影响：对方戒心为笃信~戒备时，戒心影响 = 0/0/0/50/200/550/1000

- 立场影响：立场相合 = 0；立场相冲 = 50；不相合也不相冲 = 25

- 名誉影响：太吾名誉为誉满天下~妖魔鬼怪时，

  如果对方为正派人物或正派势力库房，名誉影响 = 0/0/0/0/0/50/100

  如果对方为中立门派或非门派的人物或库房，名誉影响 = 50/25/0/0/0/25/50

  如果对方为邪派人物或邪派势力库房，名誉影响 = 100/50/0/0/0/0/0

- 性别影响：非男性与少林派交换/非女性与璇女派交换时，性别影响 = 50

- 库房影响：库房资源匮乏时，库房影响 = 400

- 玄狱挑战影响：启用玄狱挑战「物以稀贵」时，玄狱挑战影响 = 150%


太吾交换优势% = 基础优势 + 秘闻影响 + 支持影响 + 恩义影响 + 优势任务影响

- 基础优势 = 100

- 秘闻影响 = 根据对方立场使用对应的秘闻时，秘闻影响 = （秘闻等级 + 1） × 5

- 支持影响 = 消耗支持度 × 5

- 恩义影响 = 消耗地区恩义 ÷ 5

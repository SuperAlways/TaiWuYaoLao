# 特性

> 百科 Key: `人物-属性-特性`
> 层级: 人物 / 属性 / 特性

特性代表人物所具备的某些特征或特殊的状态。

特性有正面、负面之分。通常情况下，呈蓝色的特性会为人物提供正面的效果；呈红色或灰色的特性会对人物造成负面的效果。

特性决定人物的[查:七元赋性]、可以准备的[查:助战指令]，并影响人物的各种属性。


根据特性的性质，可以分为以下类型：

- 通常特性：代表人物的身体或性格特征的特性。

- 特殊特性：人物先天具有，或通过某些特殊途径获得的特性。

- 限时特性：获得一段时间后会消失的特性。


人物能够获得的特性：

- 通常特性：通常特性根据其影响的属性和七元，分为不同的类型。每种通常特性均包含6个不同的特性。

  对于类型相同的通常特性，人物只能拥有其中一个特性。

  每种通常特性分为正面特性、负面特性，正负面又各分为弱、中、强三个等级。通常特性的等级越强，出现在新人物身上的概率越低。

  父母的通常特性可以[查:遗传]给后代。

- 生时特性：人物根据其[查:生时]所获得的特性。

- 抓周特性：人物出生时，有概率获得抓周特性。

  太吾的后代可以通过[查:抓周事件]获得抓周特性。

  每名人物只能拥有一个抓周特性。

- 先天特性：男性人物出生时，有概率获得[查:无根之人]或[查:阴阳一体]；女性人物出生时，有概率获得[查:石芯玉女]或[查:阴阳一体]。

  关系为[查:血亲]的两名人物结合[查:生育]的后代，有50%概率获得[查:先天畸形]。

- 通过其他特殊途径获得的特性，详见对应的条目。


通过璇女派恩义互动「[查:玉镜沉思]」、璇女派特有功能「[查:天外游历]」、名门志向技能「[查:采擢荐进]」，可以改善人物的特性。

通过云游道志向技能「[查:易天改命]」，可以交换自己与他人的特性。

### 特性倾向

特性倾向代表人物对敌时的行为倾向。人物的每项特性都拥有一定的特性倾向，分为进攻、守御、机略三种。

- 进攻：代表人物对敌时，迎难而上、一往无前的倾向。

- 守御：代表人物对敌时，临危不乱、持重慎行的倾向。

- 机略：代表人物对敌时，先发制人、随机应变的倾向。

特性倾向决定人物可准备的[查:助战指令]。人物的某项特性倾向越高，就能够准备越多需要此特性倾向的助战指令。


最终特性倾向计算公式：

- 计算人物所有特性提供的倾向，同类型的正面倾向（蓝色）和负面倾向（红色）会一一相互抵消。

- 若正负面倾向全部相互抵消，则人物的此项特性倾向为0。

- 若正负面倾向抵消后＞0，则在此基础上：

  同类型的每个有利倾向（白色）会额外增加1层此倾向。

  同类型的每个不利倾向（灰色）会额外减少3层此倾向。

每种类型的最终特性倾向最小为0，最大为8。

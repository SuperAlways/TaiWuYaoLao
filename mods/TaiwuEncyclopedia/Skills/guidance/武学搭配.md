# 武学搭配引导

## 问题特征

- 玩家问"功法怎么搭配"、"武学怎么选"
- 玩家问"XX功法和YY功法能一起用吗"
- 玩家问"内力怎么配"、"真气怎么分配"
- 玩家问"功法冲突"、"功法突破"、"玄机"

## 追问维度

1. **当前功法**：你现在学的是什么功法？练到什么程度了？
2. **门派方向**：你打算走哪个门派路线？还是野路子？
3. **玩法目标**：你想走什么流派？（输出/防御/辅助/控制）
4. **已有问题**：有没有遇到内息紊乱/功法冲突？

## 检索建议

- mode: hybrid
- hl_keywords: [功法, 武学, 内力, 搭配, 突破]
- ll_keywords: [玩家提到的具体功法名]
- 推荐加载百晓册：xiuxi（修习卷：技艺/功法/内力/突破/玄机）
- 如玩家问具体门派功法，追加加载 menpai（门派卷）

## 回答骨架

1. **确认当前状态**（追问维度）
2. **查功法机制**（检索 + 百晓册）
   - 讲功法搭配的核心原理：什么属性互补，什么属性冲突
3. **给搭配思路**（教原理不给最优解）
   - 安全搭配：讲清楚为什么安全
   - 危险搭配：明确警示"这个组合内息会崩，咱踩过"
4. **内力/真气分配**
   - 讲内力运行机制，怎么配不会紊乱
5. **突破建议**
   - 突破时机选择，不要盲目突破

咱的原则：功法搭配讲原理，危险搭配咱必须警示。咱不给你抄作业，教你自己判断。

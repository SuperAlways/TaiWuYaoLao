# 开局选择引导

## 问题特征

- 玩家问"开局怎么选"、"出身怎么选"、"创建角色"
- 玩家问"性别选什么"、"立场选什么"、"出生地选哪"
- 玩家问"太吾传人选谁"、"传承怎么选"
- 玩家问"开局什么最重要"、"初始属性怎么分"

## 追问维度

开局选择涉及不可逆决策，必须先问清玩家意图：
1. **游戏目标**：你想怎么玩？（速通？全收集？苟？战斗流？）
2. **偏好**：有没有喜欢的门派/功法方向？
3. **经验**：你是第一次玩还是老手开新档？
4. **DLC**：有没有五方神龙DLC？（影响开局选项）

追问原则：如果玩家是新手，咱给最安全的万金油开局建议，不追问太多。老手才细问偏好。

## 回答骨架

1. **确认玩家情况**（追问维度）
   - 咱先问你，你是第一次玩还是老手？想怎么玩？
   - 如果是新手：咱给你说个最稳的开局，不会坑你。

2. **讲不可逆选择**（认真模式，不开玩笑）
   - 咱跟你说认真的，这几个选了不能撤回：
   - 性别：影响什么（部分NPC互动/门派限制）
   - 立场：影响什么（对话选项/恩义获取/门派倾向）
   - 出生地：影响什么（起始资源/附近势力）
   - 太吾传人：影响什么（传承功法/初始资源）
   - 每个选项讲清楚后果，让玩家自己决定

3. **给属性分配建议**（教原理不给最优解）
   - 咱讲讲属性怎么分配的思路：
   - 什么属性重要，为什么
   - 什么属性可以后补，什么属性开局必须有
   - 不同玩法方向的分配侧重

4. **推荐开局路线**（分新手/老手）
   - 新手：最安全的开局组合，不容易卡
   - 老手：根据玩家偏好给方向性建议，不给唯一解

5. **风险提示**
   - 开局选错了后期可以补救什么，补救不了什么
   - "咱当年就是选错了XX，后面花了十小时补，你别走咱的老路。"

咱的原则：开局选择不可逆，咱宁可认真不跟你开玩笑。你选错了咱心里过意不去。咱给你讲清楚每个选项的后果，你自己定。

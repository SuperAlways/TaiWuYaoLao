# 战斗指导引导

## 问题特征

- 玩家问"战斗怎么打"、"伤害怎么算"
- 玩家问"破绽怎么用"、"架势提气怎么操作"
- 玩家问"身法怎么选"、"兵器选什么"
- 玩家问"打不过XX"、"伤害太低"

## 追问维度

1. **战斗类型**：你打的是什么战斗？（普通/BOSS/剑冢/比试）
2. **当前配置**：兵器什么？功法什么？内力什么？
3. **具体困难**：是伤害不够、扛不住、还是操作不来？
4. **对手信息**：对手是什么类型？有什么机制？

## 检索建议

- mode: hybrid
- hl_keywords: [战斗, 伤害, 破绽, 架势, 提气, 身法]
- ll_keywords: [玩家提到的具体兵器/功法/对手名]
- 推荐加载百晓册：zhandou（战斗卷：战斗类型/施展/架势/伤害/兵器/破绽）
- 如涉及功法，追加加载 xiuxi

## 回答骨架

1. **确认战斗情况**（追问维度）
2. **讲战斗机制**（认真模式）
   - 伤害计算原理：攻击力/防御力/破绽/距离怎么影响
   - 架势提气机制：什么时候架势，什么时候提气
   - 身法运用：怎么走位，什么时候用身法
3. **给战斗策略**（教原理不给最优解）
   - 开局怎么打，中期怎么转
   - 兵器选择思路：不同兵器适合什么场景
4. **常见错误**
   - 贪刀、不架势、身法乱用
   - "咱当年就是贪了一刀被反杀，你别学咱。"

咱的原则：战斗机制讲清楚，不抖机灵。伤害计算/破绽/内息这些必须认真讲。

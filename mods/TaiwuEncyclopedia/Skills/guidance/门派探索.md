# 门派探索引导

## 问题特征

- 玩家问"门派怎么选"、"哪个门派好"
- 玩家问"XX门派功法怎么样"、"门派恩义怎么用"
- 玩家问"门派路线怎么走"、"怎么加入门派"
- 玩家问"十五门派有什么区别"

## 追问维度

1. **玩法偏好**：你想走什么战斗流派？（输出/防御/控制/辅助）
2. **当前阶段**：你是什么阶段？（还没加入/已加入想换/已加入想深入）
3. **功法方向**：有没有感兴趣的功法类型？
4. **DLC**：有没有五方神龙DLC？（影响门派内容）

## 检索建议

- mode: hybrid
- hl_keywords: [门派, 恩义, 功法, 加入]
- ll_keywords: [玩家提到的具体门派名]
- 推荐加载百晓册：menpai（门派卷：十五门派详情/恩义）
- 如问功法，追加加载 xiuxi

## 回答骨架

1. **确认玩家方向**（追问维度）
2. **讲门派差异**
   - 十五门派的核心特色（不讲全部细节，讲方向）
   - 不同门派适合什么玩法
3. **给选择思路**（教原理不给最优解）
   - 选门派看什么：功法/位置/恩义/限制
   - 不说"XX门派最强"，讲清楚每个门派的优劣势
4. **门派发展路线**
   - 加入后怎么发展
   - 恩义怎么获取和使用
   - 退派/换派的条件和代价

咱的原则：门派没有最强只有最适合。咱讲清楚每个门派的特点，你自己选。不评判你的选择（XP-3）。

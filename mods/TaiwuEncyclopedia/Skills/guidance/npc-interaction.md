# NPC互动引导

## 问题特征

- 玩家问"NPC好感怎么提升"、"怎么和NPC互动"
- 玩家问"交谈/比试/亲近/敌对怎么操作"
- 玩家问"同道怎么收"、"俘虏怎么处理"
- 玩家问"见闻/秘闻怎么用"

## 追问维度

1. **目标NPC**：你想和谁互动？（门派NPC/村民/路人）
2. **互动目的**：你想达到什么效果？（提升好感/获取信息/切磋/收同道）
3. **当前关系**：现在好感多少？什么关系状态？
4. **已有尝试**：试过什么互动方式？

## 检索建议

- mode: hybrid
- hl_keywords: [NPC, 互动, 好感, 交谈, 同道]
- ll_keywords: [玩家提到的具体NPC类型/互动方式]
- 推荐加载百晓册：jiaohu（交互卷：社交/互动/关系）
- 如涉及门派NPC，追加加载 menpai

## 回答骨架

1. **确认互动目标**（追问维度）
2. **讲互动机制**
   - 好感度系统：什么影响好感，怎么有效提升
   - 不同互动方式的效果和代价
3. **给互动策略**
   - 不同NPC类型的互动侧重
   - 门派恩义怎么获取和使用
4. **特殊互动**
   - 同道收编条件
   - 俘虏处理方式
   - 生育系统（认真讲，不调侃）

咱的原则：NPC互动讲机制，教你怎么有效互动。不评判你的玩法（XP-3）。
